## 《看房笔记》PWA 组件级 UI 规格（可直接实现）

风格基准：参考 Notion 的内容块式布局与克制视觉。

---

# 1. 全局尺寸系统（Spacing System）

| 级别  | 数值   | 用途         |
| --- | ---- | ---------- |
| XS  | 4px  | 图标与文字微距    |
| S   | 8px  | 输入内边距、标签间距 |
| M   | 12px | 模块内部元素间距   |
| L   | 16px | 页面左右边距、行间距 |
| XL  | 24px | 模块之间间距     |
| XXL | 32px | 页面分区大间隔    |

---

# 2. 圆角与边框

| 元素    | 圆角    | 边框          |
| ----- | ----- | ----------- |
| 卡片    | 12px  | 1px #EAEAEA |
| 按钮    | 10px  | 无           |
| 标签    | 999px | 无           |
| 输入框底线 | 0     | 1px #EAEAEA |

---

# 3. 按钮系统

## 3.1 主按钮（Primary）

用途：页面唯一主行动作

* 高度：44px
* 内边距：0 16px
* 背景：#2F80ED
* 文字：#FFFFFF，14px，600
* 圆角：10px
* 阴影：无

状态：

* Hover：背景加深 8%
* Disabled：背景 #BFD7FF，文字 #FFFFFF

---

## 3.2 次级按钮（Secondary）

* 高度：40px
* 边框：1px #EAEAEA
* 背景：#FFFFFF
* 文字：#222
* 圆角：10px

---

## 3.3 文本按钮（Text）

* 颜色：#2F80ED
* 字号：14px
* 无背景

---

# 4. 列表卡片（房源卡）

尺寸：

* 内边距：16px
* 图片：72×72px，圆角 8px
* 卡片间距：12px

结构：

```
[72px 图]   房源标题（16px/600）
            次行信息（12px/灰）
            标签行（S间距）
右上角：总分（22px/600）
```

点击反馈：

* 背景闪烁 #F7F7F7 80ms

---

# 5. 输入控件规范

## 5.1 文本输入

* 高度：36px
* 下边框：1px #EAEAEA
* Focus：下边框 #2F80ED

Placeholder：#999

---

## 5.2 多行输入（速记）

* 最小高度：72px
* 内边距：12px
* 背景：#FAFAFA
* 圆角：8px

---

## 5.3 滑杆（评分 & 权重）

* 轨道高度：2px
* 轨道色：#E0E0E0
* 已选轨道：#2F80ED
* 滑块：14px 圆点
* 数值显示：右侧 12px 灰字

---

## 5.4 开关（Switch）

* 使用系统原生样式
* Off：#DADADA
* On：#2F80ED

---

## 5.5 标签（Tag）

* 高度：24px
* 内边距：0 10px
* 字号：12px
* 正向：背景 #EAF6EF，字 #2E7D32
* 风险：背景 #FDECEA，字 #C62828
* 中性：背景 #F2F2F2，字 #666

---

# 6. 页面级布局细化

## 6.1 顶部导航栏

* 高度：48px
* 左右内边距：16px
* 标题：16px，600
* 图标尺寸：20px

---

## 6.2 底部主按钮（录入页）

* 固定底部
* 距底 16px
* 宽度：calc(100% - 32px)

---

## 6.3 模块块（Block）

用于录入页：

```
Block 标题（12px 灰字）
内容区（M间距）
底部分割线
```

Block 之间间距：24px

---

# 7. 图标系统

风格：线性、1.5px stroke
尺寸：20px

建议图标集：Feather / Lucide（轻量、克制）

---

# 8. 动效规范

| 场景           | 时长    | 类型                          |
| ------------ | ----- | --------------------------- |
| 页面切换         | 180ms | ease-out                    |
| 列表重排         | 200ms | ease                        |
| Modal/Drawer | 220ms | cubic-bezier(0.2,0.8,0.2,1) |

禁止弹跳、夸张动画。

---

# 9. 暗色模式（可选 v1.2）

| 元素   | 暗色      |
| ---- | ------- |
| 背景   | #121212 |
| 卡片   | #1E1E1E |
| 主文字  | #FFFFFF |
| 次级文字 | #AAAAAA |
| 分割线  | #2A2A2A |

---

# 10. 视觉验收Checklist

* 页面可视元素不超过 3 层视觉层级
* 不使用多种品牌色
* 所有输入控件对齐到同一左边线
* 模块间距统一为 24px
